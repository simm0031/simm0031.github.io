  1. Qus 1: What are the three components of Web?

    Ans:

    • HTML
    • CSS
    • JavaScript

  2. Qus 2: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. GitHub is an example of this.

  3. Qus 3: What tag is used to create the title of a web page? Give an example.

    Ans: Head tag. <title> this is the title </title>

  4. Qus 4: What tag is used to create a paragraph in html? Give an example.

    Ans: P tag. <p> this is a paragraph </p>

  5. Qus 5: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: <a>, because it's the only attribute tag. the others, I believe are elements.

  6. Qus 6: What are the tags required to create a navigation list in HTML?

    Ans: <nav>, and <ul>, <ol>, and/or <dl>, using <li> for each list item

  7. Qus 7: What element is used to create images? Name its attributes.

    Ans: <img>, which has attributes for width, height, source (src), and alternate text (alt) .

  8. Qus 8: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Ans: <b>, because the others are all for tables.
